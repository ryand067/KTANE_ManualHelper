Version 1.1

This is a small minor update completed on December 31, 2020. It modifies the color scheme of the Clear Button and changed the Divisibility Mod Modules's second number to radio buttons instead of another textbox

New Modules:

  • Password Helper

New Features:

  • Changed the Hover color of the Clear Button to be orange instead of the blue it currently was to match the rest of the color scheme
  • Changed the second textbox in the Divisibility Mod Module section to be radio buttons instead.
  • Modified links in the help page to make them open in new tabs
  • Filled in the missing How to use section for the Jack-o-lantern Mod Module
  • Added a Search prediction to the 1,000 Words Mod module and to the new Passwords Helper Module

Version 1.0

Initial Version

This is the initial version since the Version History Page was created.

New Modules:

  • Memory Helper
  • Sequence of Wires Helper
  • Complicated Wires Helper
  • Addition
  • 1,000 Words
  • Crazy Talk
  • Chicken Nuggets
  • Modulo
  • Divisible Numbers
  • Jack-O'-Lantern

New Features:

  • Appendix A: Tables

    • Used for viewing tables when modules require huge tables that wouldn't make sense to duplicate in HTML.

    Including Table 1: Crazy Talk, Table 2: Divisibility Table, and Table 3: Jack-O'-Lantern Table.

  • A clear button to clear everything.
  • The ability to press enter while on a regular text box to submit the numbers and text in the helper functions.
  • A new Footer Thing at the bottom of the Helper Modules section for Help and Navigation to other pages.
