Version 1.2

Date Finished: March, 2021

Version 1.2 contains 2 brand new Modded Modules as well as an interactive sub-site to help with using one of the new modded modules.

New Modules:

  • Bone Apple Tea Module Helper
  • Clue (Murder) Module Helper
  • Clue (Murder) Interactive Table Sub-Page Helper

New Features:

  • New Bone Apple Tea Modded Module
  • New Clue (Murder) Modded Module
  • Interacve mini sub-page for clickable table in the Clue (Murder) Modded Module

Version 1.1

This is a small minor update completed on December 31, 2020. It modifies the color scheme of the Clear Button and changed the Divisibility Mod Modules's second number to radio buttons instead of another textbox

New Modules:

  • Password Helper

New Features:

  • Changed the Hover color of the Clear Button to be orange instead of the blue it currently was to match the rest of the color scheme
  • Changed the second textbox in the Divisibility Mod Module section to be radio buttons instead.
  • Modified links in the help page to make them open in new tabs
  • Filled in the missing How to use section for the Jack-o-lantern Mod Module
  • Added a Search prediction to the 1,000 Words Mod module and to the new Passwords Helper Module
  • Added a Search function to the CrazyTalk
  • Added a Search Prediction the that CrazyTalk functionality
  • Added tool-tips to the complicated Wires table to display the letter's description when hovering over it.
  • Added a feature to the Addition to ignore empty lines so if the user accidently includes an extra line at the end it doesn't display an error, making the user have to type in all of the numbers again.
