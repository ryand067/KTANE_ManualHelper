Memory

This Module-Helper is used for the Memory Module from the original game.

Reason: The reason this is a helper module is because it can be hard to remember all of the different numbers and positions for each step. This helper-module seeks to help with that issue.

How to use

To use this module, start with the first step. Whatever the number pressed was, record that in the text-area first, followed by whatever position that number was in without spaces, then press "Enter".

Example: If the instructions say for step 1 press the button in the 2nd position, you'd ask the diffuser what button is in the 2nd position. Let's pretend it is the number 4. In this example, you'd input the number 42 into the text-area, and press enter. Because the 4 in 42 represents the number pressed and the 2 in 42 represents the 2nd position.

Keep on doing this for all steps. If you need to look back to see what you pressed in the 2nd stage, for example, just look at the 2nd 2-digit number and relay the information to your diffuser.


Sequence of Wires

This Module-Helper is used for the Sequence of Wires Module from the original game.

Reason: The reason this is a helper module is because it can be hard to remember all of the different number of each colored wire. This helper-module seeks to help with that issue.

How to use

To use this helper-module, everytime the diffuser tells you what color wire they see, press the increase button to the right of that color's text-field. This will automatically increase the number which is 0 by default.

That way, before you look at the table in the original manual section, you look at the text-field for that particular wire to see what number of that colored wire you are on.

Example: For example, if you currently have 3 for red, 2 for blue, and 1 for black, and the user tells you they have a black wire that goes to C, you would press the up arrow to the right of the black text-field. This will increase the black number to 2. Now, you will look at the original bomb-game manual and see if the manual says to cut the second black wire or not and relay the information back to your diffuser.


Password Helper

This Module-Helper is used for the Password Module from the original game.

Reason: The reason this is a helper module is because sometimes it can be hard to look through the big list of valid passwords in enough time. This way, You enter a few letters and all possible valid passwords appear instantly.

How to use

Start typing in the first few letters of a password you think might work. A list of valid passwords with the same prefix will appear. If the password you're thinking of isn't in the list, then it's not a valid password. Otherwise, it is and continue to guess more letters.


Complicated Wires

This Module-Helper is used for the Complicated Wires Module from the original game.

Reason: The reason this is a helper module is because it can be hard to interpert that massive venn-diagram on the original manual; which can be bad when under a very tight time constraint. This helper-module seeks to help with that issue.

How to use

The user will tell you a wire complication. For example, a plain white wire with a star, but no L.E.D.

Find this information in the table and look at the letter. Using the 2nd table, relay the information to your diffuser.


Addition

This is a modded module. It doesn't come with the original game. The original Manual can be found Here.

How to use

In the addition module, there will always be 10 numbers to sum together. The text-area is designed to fit exactly 10 numbers with no scroll bar. If you see a scroll bar, you have inputted too many numbers.

The diffuser will read you each number one at a time. Type all ten numbers into this text-area. When all 10 numbers have been entered, press the "GO" button below.

If you see an error message in the result box, you have inputted a non-number, try again re-entering all numbers. If all numbers were inputted correctly, it should tell you the sum. Relay this answer to your diffuser.


Crazy Talk

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

To use this module, the user will tell you a phrase. Using Appendix A: Tables, Table A: CrazyTalk, find this phrase on the table or search for it with the Search Functionality Enter Arrows as < and >. Once the phrase is down, tell the diffuser the numbers needed for pressing the switch down, then up.


Modulo

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

To use this mod, the diffuser will tell you a big number and a small number. Big meaning larger in text, not value. You will then enter the bigger number (the first number) into the first text box and the smaller number (the second number) into the second text box. Then, either press "Enter" or click "GO" to submit. The function will automatically perform the operation and tell you the result in the box below. Relay the result of the modulos operation to your diffuser.


Chicken Nuggets

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

The diffuser will tell you the number of chicken nuggets a customer is requesting. For example, 67 nuggets. Click the radio button for one solution only or all solutions. Sometimes, there will be a ridiculous number of boxes, requiring the diffuser to click a button numerous times, to avoid this, there is the option to look at all solutions to try and find one with minimal button presses required. Next, enter the number into the text-box and either click "GO" or press "Enter". The results will be shown in the box below. Relay the results to the diffuser.


1,000 Words

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

There are 5 different words for each iteration of this module. One at a time, the diffuser will tell you a word, or more like sequence of letters since most don't spell out actual English words, and you enter them into the text-box provided. Then, either press "Enter" or click "GO" to submit. The results will be shown in the box below in the form of yes the word was found, or no it wasn't. Relay this information back to your diffuser.


Divisible Numbers

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

The diffuser will give you a number. Place this number in the first provided text-box. Based on information from the bomb like number of batteries etc., you will get a second number. Select the correct radio button based on which rule matched your bomb and press "Enter" or click "GO". The results will be displayed in the box below in the form of "YEA" or "NAY". Relay this information to your diffuser, and repeat 2 more times. Note: the 2nd number will always remain the same for each new bomb.


Jack-O'-Lantern

This is a modded module. It doesn't come with the original game. The original manual for this mod can be found Here.

How to use

To use this mod module, the diffuser will tell you the shape of the Jack-O'-Lantern's eyes and mouth and if there's a nose or not. From there, using the shape of the eyes and mouth, the user will ask the diffuser if some item on the table is present on their bomb. For Example: Unlit Indicators > Lit indicators. If this is true then the diffuser should press Trick, otherwise, they should press Treat. If, however, there is a nose, the the buttons are reversed.
